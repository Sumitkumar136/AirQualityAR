# AirQualityAR - See the Air You Breathe
AirQualityAR is a web application that transforms invisible air pollution data into immersive, actionable insights powered by AI. It provides real-time air quality information, predictive analytics, and an augmented reality (AR) experience to help users understand the air they breathe.

---

# üåü Features
- Real-Time Air Quality Dashboard: Get live pollution data from monitoring stations worldwide
- Interactive Map View: Explore air quality data from monitoring stations around the world
- Immersive AR Experience: Point your phone at your surroundings to see a 3D pollution heatmap
- 48-Hour Pollution Forecast: Our predictive models analyze traffic, weather, and industrial activity to forecast pollution spikes
- AI-Powered Insights: Machine learning models analyze air quality patterns to provide deeper insights and predictions
- Health Impact & Safety Guidelines: Understand how different pollution levels affect your health and what precautions you should take

 # üìä API Configuration
The application uses the World Air Quality Index (WAQI) API to fetch real-time air quality data. The API key is included in the code, but for production use, it's recommended to:

Get your own API key from WAQI
Replace the existing API key in the index.html file:
```
const API_KEY = 'your_api_key_here';
```

# üéØ Usage
1. View Air Quality: Enter a city name in the search box to get live pollution data
2. Map View: Click on any location on the map to see air quality data for that area
3. AR View: Click the "Launch AR View" button to experience the AR visualization (requires camera access)
4. Analytics: View the 48-hour pollution forecast for the selected location
5. ML Insights: Explore AI-powered insights including anomaly detection, pollution source prediction, and health recommendations
6. Health Impact: Learn about how different pollution levels affect your health and what precautions to take
